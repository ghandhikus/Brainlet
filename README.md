# Brainlet
Blackbox based AI for games. Provides multiple of solutions for things like mood, goal picking and identity.

## Brainlet Usage
```java
    // To use brainlet you need to override Brainlet class
    // 
    // Brainlet core parts:
    // 
    // - Long-term memory
    // - Short-term memory
    // - Sensorics
    // - Identity
    // - Mood
    // - Decision-making blackbox
    // 
    //
    // Additional information:
    // - For long-term memory you should use database that won't use RAM to avoid hauling megabytes of data.
    // 
    // 
```
